# bookingENI
bookingEni est une application web pour la réservation de salle (visioconférence, salle de cours) et d'équipements (PC portables, vidéo projecteurs, kits audio, etc..) à l'ENI

## Installation 
Use Python 3.14.2,
Use PIP 25.3,
Use Django 6.0.2

Go to the project directory
```bash
cd bookingEni
```

Create vitrual environment venv
```bash
python -m venv .venv
```

Activate virtual environment (Windows)
```bash
.venv\Scripts\Activate.ps1
```
Install dependencies
```bash
pip install -r requirements.txt
```

Install Django
```bash
pip install Django
```

Starting the development server
```bash
python manage.py 
```
